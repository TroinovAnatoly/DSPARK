# Naming Guidelines

## Components
PascalCase
Example: ItemCard, ShopPage

## Hooks
camelCase + prefix use
Example: useCart, useGames

## Redux
slices: featureNameSlice
thunks: featureNameThunks

## Files
- components: PascalCase
- utils: camelCase
- slices: camelCase + Slice
- thunks: camelCase + Thunks

## Constants
SCREAMING_SNAKE_CASE

## API
camelCase functions
